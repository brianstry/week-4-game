Crystal Collector!

instructions

You will be given a number at the start of the game

There are four crystals below. By clicking on a crystal, you will add a specific amount of points to your own number.

You will win the game by matching your own number to the random number generated at the begining of each round. But be carefule, if you exceed the number given, you lose.

The value of each crystal will change each round. So have fun!
